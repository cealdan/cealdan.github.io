Privacy Policy ‚Äì AutoXic

Last updated: November 18, 2025

The AutoXic app uses Android's Accessibility Service to perform a magic trick as described in the app tutorial. This policy explains what data is accessed, how it is used, and the user's rights.

1. Data Accessible via the Accessibility Service

When the user enables the Accessibility Service (necessary for the trick to work), AutoXic can access only:

  • the calculation result displayed on the calculator,
  • the calculator interface elements required for the trick.

The app then uses this information to automatically press certain calculator buttons when the user presses the volume buttons, as described in the tutorial.

No other information is read, stored, or transmitted.

AutoXic never collects:

  • messages, emails, photos, videos,
  • identifiers, contacts, passwords,
  • sensitive or private data,
  • content from other apps.

2. Data Not Collected

AutoXic:

  • does not store any data on the device,
  • does not send any data to external servers,
  • does not create anonymous or advertising IDs,
  • does not collect any personal information.

All processing is done locally on the user's device.

3. Use of Accessible Data

The data read on the screen is used only to:

  • retrieve the calculation result,
  • perform the automated actions of the magic trick (simulated button presses),
  • display the final result expected by the routine.

Data is never stored after the trick is executed.

4. Data Sharing

AutoXic:

  • does not share any data with third parties,
  • does not sell any data,
  • does not transmit any information outside the device.

5. Security

Since no data is stored or transmitted, the risk of exposure is extremely low. AutoXic operates entirely offline.

6. User Rights

The user can:

  • disable the Accessibility Service (but the trick will no longer function),
  • uninstall the app to completely remove its functionality,
  • contact the developer with any questions.
